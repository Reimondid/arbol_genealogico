# 🌳 Árbol Genealógico Dinámico

Aplicación web interactiva para crear y visualizar árboles genealógicos de forma dinámica.

## 🚀 Características

- Visualización interactiva con arrastre y zoom
- Edición de personas y relaciones familiares
- Carga y exportación de datos en formato CSV
- Búsqueda de personas en tiempo real
- Cálculo automático de edades
- Soporte para múltiples matrimonios
- Interfaz oscura y moderna

## 📦 Lenguajes

- HTML5
- CSS3
- JavaScript (Vanilla)
- [vis-network](https://visjs.github.io/vis-network/) - Visualización de grafos
